# ⚠️ Known Limitations

Beberapa batasan yang disadari:

- Tidak memperhitungkan kondisi cuaca aktual
- Rukyat berbasis imkan, bukan laporan lapangan
- Metode global mengikuti kalender baku, bukan observasi lokal
- Perbedaan otoritas falak antar negara tidak diseragamkan

Dokumen ini ada untuk menegaskan:
proyek ini **jujur secara ilmiah**.
