# ⚠️ Known Limitations

Beberapa batasan yang disadari:

- Tidak memperhitungkan kondisi cuaca aktual (awan, hamburan cahaya horison).
- Rukyat berbasis imkan (prediksi visibilitas), bukan laporan lapangan visual.
- Metode global mengikuti kalender baku sesuai kesepakatan internasional (Turki 2016).
- Perbedaan otoritas falak antar negara tidak diseragamkan (API hanya menyediakan komparasi).
- Ketelitian waktu terbatas pada resolusi ephemeris NASA SPICE dan algoritma bisection search (±1 detik).
- Koreksi refraksi atmosfer menggunakan model standar (diabaikan pada kriteria geosentris murni).

Dokumen ini ada untuk menegaskan:
proyek ini **jujur secara ilmiah**.
